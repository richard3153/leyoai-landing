# 杭州市上城区乐友信息服务工作室 - 公司运营手册 v2.1

## 公司概览
- **公司名**: 杭州市上城区乐友信息服务工作室（个体工商户）
- **品牌**: LeyoAI
- **定位**: MaaS (Model as a Service) 平台
- **使命**: 让 AI 模型服务每个人
- **愿景**: 成为全球领先的垂直领域 AI 模型服务商

## 四大产品线

| 产品 | 名称 | 状态 | 说明 |
|------|------|------|------|
| 🛡️ Cyber Model | 网络安全模型 | ✅ 已上线 | 威胁检测、漏洞分析、安全合规 |
| 🎬 Video Model | 视频安全模型 | ✅ 已上线 | 视频安全审核、内容合规分类 |
| ⚙️ Flow Model | 流程自动化模型 | ✅ 已上线 | 流程编排、Python 脚本生成 |
| 📊 Analytics Model | 数据分析模型 | ✅ 已上线 | A/B 测试、数据可视化 |

## 技术栈
- **前端**: React + Vite + TypeScript + TailwindCSS
- **部署**: Vercel + GitHub Actions
- **模型**: Qwen2.5 + LoRA 微调
- **数据库**: Supabase
- **AI Space**: HuggingFace Spaces (Streamlit)

## 合规原则
- 所有模型需确保训练数据版权合规
- 发布前完成专利自查
- 遵守 HuggingFace/GitHub 平台政策及当地法规
