数据存储与安全

Data Storage & Security

我们采取严格的安全措施保护您的数据：

We implement strict security measures to protect your data:

  • 传输加密：所有数据传输均采用 HTTPS/TLS 加密协议
  • Transmission Encryption: All data transmission uses HTTPS/TLS encryption protocols
  • 存储安全：敏感数据采用加密存储，访问严格控制
  • Secure Storage: Sensitive data is encrypted with strict access controls
  • 访问控制：仅授权人员可在必要时访问用户数据
  • Access Control: Only authorized personnel can access user data when necessary
  • 安全审计：定期进行安全评估和漏洞检测
  • Security Audits: Regular security assessments and vulnerability detection
  • 数据保留：对话记录在服务完成后按策略定期清理
  • Data Retention: Conversation records are regularly cleaned according to policy after service completion
