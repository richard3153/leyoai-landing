## 一、产能定义与衡量指标

### 1.1 什么是"产能"

在 Nexify AI 模型研发场景下，**产能**定义为：单位时间内，团队能够高质量交付的 AI 模型研发工作量。

**产能构成要素：**

| 维度 | 定义 | 计量单位 | 说明 |
|------|------|----------|------|
| **GPU 算力** | 可用于模型训练的 GPU 资源 | GPU·小时/天 | 核心瓶颈资源 |
| **人力产能** | 算法工程师的有效工作时间 | 人·天/周 | 含数据清洗、模型调优、评测 |
| **模型产出** | 可发布的模型数量 | 个/月 | 需通过质量门槛 |
| **用户请求** | 可处理的推理请求量 | 次/天 | 含 API 调用和内部测试 |

**核心公式：**
```
有效产能 = min(GPU 算力产能, 人力产能, 基础设施容量)
```

---

### 1.2 产能利用率计算公式

**GPU 利用率**
```
GPU 利用率 = 实际训练时间 / 可用 GPU 时间 × 100%
```

**综合产能指数 (CPI)**
```
CPI = (GPU 利用率 × 0.4) + (人力利用率 × 0.3) + (交付效率 × 0.3)

分级标准：
- CPI ≥ 85%：优秀
- 70% ≤ CPI < 85%：健康
- 55% ≤ CPI < 70%：亚健康
- CPI < 55%：严重问题
```

---

### 1.3 产能预警阈值

| 级别 | 触发条件 | 响应时间 | 升级路径 |
|------|----------|----------|----------|
