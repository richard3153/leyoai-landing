# Nexify 四大模型训练规划

**规划时间**: 2026-04-09
**目标**: 重新训练/新建四个垂直领域模型

---

## 一、Cyber Model（网络安全模型）

### 设计目标
- **智能威胁检测**: 威胁情报分析、攻击模式识别、APT 检测
- **漏洞分析扫描**: CVE 分析、漏洞评估、修复建议
- **安全合规检查**: 等保、ISO27001、SOC2、GDPR 合规

### 技术方案
| 项目 | 配置 |
|------|------|
| 基座模型 | Qwen2.5-3B-Instruct（升级） |
| LoRA | r=16, α=32, dropout=0.05 |
| 训练数据 | 5000+ 条 |
| 数据来源 | 自建网络安全问答 + BeaverTails |

### 数据覆盖领域
- 威胁情报（1000条）：钓鱼、恶意软件、APT、社会工程学
- 漏洞分析（1000条）：CVE、OWASP Top10、漏洞修复
- 合规审查（1000条）：等保、ISO27001、SOC2、GDPR
- 安全运维（1000条）：日志分析、入侵检测、应急响应
- 安全加固（1000条）：系统加固、网络加固、应用安全

### 预计训练时间
- Mac Studio M2 Max: ~3-4 小时
- 训练目标: Loss < 0.3

## 二、Video Model（视频内容模型）

### 设计目标
- **视频安全分析**: 视频内容安全检测、风险识别
- **内容理解**: 场景识别、物体检测、行为分析
- **智能审核**: 违规内容识别、合规判断
- **智能剪辑**: 场景分割、关键帧提取、精彩片段

### 技术方案
| 项目 | 配置 |
|------|------|
| 基座模型 | Qwen2-VL-2B-Instruct（保持） |
| LoRA | r=16, α=32, dropout=0.05 |
| 训练数据 | 5000+ 条（扩充） |
| 数据来源 | 自建 + VideoMAF + UCF-Crime |

### 数据覆盖领域
- 视频场景分析（1500条）：多场景识别
- 内容审核（1500条）：违规检测、合规判断
- 安全分析（1000条）：危险行为、异常检测
- 智能剪辑（500条）：场景分割、关键帧
- 交通安全（500条）：新增领域

### 预计训练时间
- Mac Studio M2 Max: ~2-3 小时

## 三、Flow Model（自动化流程模型）

- **任务调度**: 定时任务、依赖调度、重试机制
